# 🚀 AI Study Chatbot (Offline)

An AI chatbot that runs completely offline using Ollama (TinyLLaMA) and Streamlit.

## 🔥 Features
- 💬 Chat with AI locally (no API cost)
- 🧠 Conversational memory
- 📚 Helps with DSA, DBMS & interview prep

## 🛠️ Tech Stack
- Python
- Streamlit
- Ollama (TinyLLaMA)

## ▶️ Run Locally
pip install -r requirements.txt  
streamlit run app.py
